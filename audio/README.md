# 音效目录

把以下音效文件放到本目录即可在游戏中播放（建议 MP3，体积小、兼容好）：

| 文件名   | 用途           |
|----------|----------------|
| hit.mp3  | 普攻/技能命中  |
| kill.mp3 | 击杀怪物       |
| levelup.mp3 | 升级选技能  |
| buy.mp3  | 商店购买       |
| hurt.mp3 | 玩家受伤       |

- 若某文件不存在，对应时机不会播放，不会报错。
- 想加新音效：在 `main.js` 的 `SOUND_URLS` 里增加一项，并在需要处调用 `playSound('新名字')`。
