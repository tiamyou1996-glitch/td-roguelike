# 怪物血量公式对比（均按「第 20 波 = 5000 血」校准）

所有公式在 20 波时均为 5000 血，第 1 波均为 15（软上限除外，见说明）。便于横向对比不同曲线形状。

---

## 一、公式与常数

| 类型 | 公式 | 说明 |
|------|------|------|
| **指数** | `15 × 1.37^(wave-1)` | 每波 ×1.37，原版 1.5 的温和版 |
| **幂函数** | `15 × wave^2.176` | 当前用的那种，前平后陡 |
| **线性** | `15 + (wave-1) × 262.37` | 每波加固定值 |
| **平方根** | `15 + 1434×(√wave - 1)` | 前期涨得快，后期变慢 |
| **对数** | `15 × (1 + 110.9×ln(wave))` | 前期极陡，后期几乎不涨 |
| **混合(线+幂)** | `2.63×wave + 12.37×wave²` | 线性 + 二次项，拐点可调 |
| **软上限** | `29925 × wave² / (wave² + 1994)` | 趋向 5000 上限，S 形 |
| **二次型** | `15 + 13.8×(wave-1)²` | 每波“加的量”递增，无一次项 |

## 三、曲线特点简述

| 类型 | 前期 | 中期 | 后期 | 适合感觉 |
|------|------|------|------|----------|
| **指数** | 平 | 渐陡 | 很陡 | 经典“越往后越难” |
| **幂函数** | 平 | 陡 | 最陡 | 前几波友好，中后压力大 |
| **线性** | 陡 | 中 | 平 | 每波加一样，最稳 |
| **平方根** | 很陡 | 中 | 平 | 前期就难，后期压得住 |
| **对数** | 极陡 | 很快平 | 几乎不动 | 前期暴涨，很快到“顶” |
| **混合** | 平 | 陡 | 很陡 | 介于线性和幂之间 |
| **软上限** | 陡 | 渐平 | 封顶 | 有明确难度天花板 |
| **二次型** | 很平 | 陡 | 很陡 | 前几波几乎不动，后面猛涨 |

## 四、若要改代码用哪一种

在 `main.js` 里目前是幂函数：

```js
const waveHp = ENEMY_MAX_HP * Math.pow(wave, ENEMY_HP_WAVE_POWER)
```

- 想换成**线性**：`waveHp = 15 + (wave - 1) * 262.37`
- 想换成**指数(1.37)**：`waveHp = 15 * Math.pow(1.37, wave - 1)`
- 想换成**二次型**：`waveHp = 15 + 13.8 * (wave - 1) * (wave - 1)`
- 想换成**平方根**：`waveHp = 15 + 1434 * (Math.sqrt(wave) - 1)`
- 想换成**混合**：`waveHp = 2.63 * wave + 12.37 * wave * wave`
- 想换成**软上限**：`waveHp = 29925 * wave * wave / (wave * wave + 1994)`
- **对数**一般不推荐（前期数值会很大）。
