**说明**：带 `【】` 的是需要你填或二选一的内容；`（ ）` 里是示例或备注。

## 一、专精与阶段（顶层结构）

- **1 层 · 愤怒化身**：集齐 **嗜血、怒击、暴怒、旋风斩** 四张后吞噬（代码已实现；暴怒另保留「累计消耗 500 怒气」单独吞噬）。
- **2 层 · 经典狂暴战**：**阶段名**，愤怒化身吞噬后**自动进入**，不抽卡。进入后抽卡池 = 三张钥匙卡 + 已选钥匙卡展开的技能。

## 四、特殊类型（非「集齐」的吞噬条件）

以下类型在链条里只需标出「条件类型 + 说明」，代码里已有对应逻辑。

| 条件类型 | 说明 | 当前例子 |
|----------|------|----------|
| 集齐 N 张 | 抽到并保留指定 N 张即激活 | 双持狂战士、愤怒化身、强化嗜血 等 |
| 60 秒战斗后吞噬 | 学到该技能起，仅战斗内计时满 60 秒 | 旋风斩、斩杀、重伤、奥丁之怒 |
| 累计获得怒气 | 选到后累计获得怒气达阈值 | 鲁莽 500、肆意放纵 1000 |
| 累计消耗怒气 | 释放某技能累计消耗达阈值 | 暴怒 500 |
| 激怒时间累计 | 处于激怒状态的累计时长达 N 秒 | 狂乱之怒等 20 秒 |
| 累计流血伤害 | 造成的流血伤害累计达 N | 浴血之躯 10000 |
| 获得即吞噬 | 选到即视为已吞噬，不占栏位 | 狂怒回复、狂怒提振、生死决战 |
| 击杀数（选到后） | 选到该技能后的击杀数达 N | （若保留） |

## 六、你补充/修改时注意

1. **专精**：若「经典狂暴战」是另一套开局或另一套核心技能，在第一节写清「激活条件 + 核心组成」。  
2. **线编号**：第二节总览里的线编号和第三节「线 1 / 线 2 / …」一致即可，方便跳读。  
3. **名称统一**：同一吞噬在文档里尽量一个名字（和代码 `SYNERGY_DEFS.name`、`ADVANCED_POOLS.poolName` 一致更好）。  
4. **SSS 等标签**：若保留「SSS鲁莽」「SSS奥丁之怒」等，可在对应线下面加一行「备注：品级/重要性 = SSS」即可。
